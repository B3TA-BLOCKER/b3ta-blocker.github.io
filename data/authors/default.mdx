## Projects

**[The Inside Job](https://github.com/B3TA-BLOCKER)** *(Final Year Project 2025–2026)*

A hardware-based red team tool built on Raspberry Pi Pico. Deploys a remote-access agent on a target machine within seconds. Engineered an encrypted C2 infrastructure on AWS using AES-256 + TLS with a Go-based in-memory agent for stealthy execution. Outperforms USB Rubber Ducky and Flipper Zero in capability.

**[Raspberry Pi IDS](https://github.com/B3TA-BLOCKER/Raspberry-Pi-based-Intrusion-Detection-System)**

A lightweight Intrusion Detection System running on Raspberry Pi. Uses an LSTM neural network trained on the CSE-CIC-IDS 2018 dataset to classify network traffic as benign or malicious in real-time. Low-power and deployable on edge hardware.

**[Anomaly Detection Using CNN-LSTM](https://github.com/danishjavedcodes/Anomalies-Detection-Using-CNN-LSTM.git)**

A video anomaly detection system using CNN-LSTM models deployed on AWS. Trained on UCF Crime, UCSD, and UBnormal datasets with Federated Learning and Particle Swarm Optimization for improved generalization.

**[GhostLogger](https://github.com/B3TA-BLOCKER/GhostLogger)**

A low-level Windows keylogger that runs invisibly in the background, capturing all keystrokes silently.
