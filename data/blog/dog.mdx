## Background

Backdrop CMS is an open-source content management system that allows users to build and manage websites. It provides a range of features and functionalities, including theme customization. However, `version 1.22.0` of Backdrop CMS`is vulnerable to an Unrestricted` 

`File Upload` issue in the 'themes' component. This vulnerability can be exploited by attackers to `achieve Remote Code Execution`.

## Exploit Overview

- **Affected**: Backdrop CMS `≤ 1.22.0`
- **Vulnerable functionality**: Theme installer (Appearance → Themes → “`Install new theme`”)
- **Issue**: `No validation on uploaded ZIP`—you can upload PHP files inside a theme and have them executed
